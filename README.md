# TaxConsulto.com

Modern, SEO-optimized tax consulting website built with Next.js 15, TypeScript, and Tailwind CSS.

## Run Locally

```bash
npm install
npm run dev
```

## Static Export

```bash
npm run build
```

The static site is generated in `out/` and is compatible with Cloudflare Pages.

## Cloudflare Pages

- Build command: `npm run build`
- Output directory: `out`
- Node version: 20 or newer
