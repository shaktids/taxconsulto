# TaxConsulto.com

A simple, fast, SEO-focused static website for TaxConsulto.com.

## Pages

- Home
- Services
- Pricing
- About Us
- Blog
- Contact Us
- Privacy Policy
- Terms & Conditions

## Cloudflare Pages Deployment

Use these settings:

- Framework preset: None
- Build command: Leave empty
- Build output directory: `/`
- Root directory: repository root

The project is plain HTML, CSS, and JavaScript, so Cloudflare Pages can publish it directly.

## Included SEO Files

- `robots.txt`
- `sitemap.xml`
- Open Graph tags
- Local Business schema
- Security and cache headers in `_headers`
- Redirect rule in `_redirects`
